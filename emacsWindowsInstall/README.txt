1.  Create a folder on the drive root.
    c:\emacs\

2. Copy the contents of the emacs-xxx-xx folder to the folder you created.

3. Add a registry key HKLU\SOFTWARE\GNU\Emacs\HOME that points to C:\emacs\.emacs.d\

4. Create the directory C:\emacs\.emacs.d\

5. Create a shortcut to C:\emacs\bin\runemacs.exe
